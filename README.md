## 使用whl安装[windows\liunx]

安装步骤:
-  whl 包在dist/文件下, 下载到本地
- ``` pip install cedar*.whl ```

## 从源码安装[liunx/windows]
windows下需要有 `sh shell` 环境

```
sh make.sh
```

## 格式化代码

```
sh black.sh
```
or
```
black -l 140 cedar
```
