## 🧠 错误模式库 (Pattern Library)
此处记录高频错误模式，AI 编码前需强制检索。

### 通用模式

- **PATTERN-001**: 路径拼接错误（跨平台兼容性）
  - **特征**: 使用 `+ '\\'` 或 `+ '/'` 拼接路径
  - **修正**: 必须使用 `os.path.join(base, sub, file)`
  - **规则**: `.cursorrules/路径处理`
  - **示例**:
    ```python
    # ❌ 错误
    path = base_dir + '\\' + filename
    # ✅ 正确
    path = os.path.join(base_dir, filename)
    ```
